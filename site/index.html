How it works

  1. Make a donation wallet Dedicated vault in Winbit32 — keep it separate from spending money.
  2. Export UFVK + address Read-only view key and unified address from the receive wizard.
  3. Walk the wizard Story, wallet, scanning plan — then pay a small ZEC quote.
  4. Share & optionally promote Send ziving.org/p/your-slug; buy homepage placement when you want eyes.

Privacy posture

  • UFVK encrypted at rest (AES-256-GCM); decrypted only in the scanner
  • No accounts, emails, or IP logging on campaign pages
  • Donation events pruned after 30 days
  • Owner token shown once — we only keep a hash
  • AI agents can set pages up via MCP on mcp.winbit32.com
